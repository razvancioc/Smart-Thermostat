\doxysubsection{Detailed Description}
ESP32 Smart Thermostat with TFT Display and Blynk Integration. 


\begin{DoxyItemize}
\item This firmware reads environmental data from a DHT sensor, controls a heating relay based on a target setpoint with hysteresis, and displays a dynamic, anti-\/flickering user interface on an SPI TFT screen. 
\end{DoxyItemize}
